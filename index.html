Apa itu Computing?

Cloud computing adalah teknologi yang memungkinkan akses dan penyimpanan data serta aplikasi melalui internet. Ini memberikan efisiensi, fleksibilitas, dan skalabilitas bagi pengguna pribadi maupun bisnis.

Manfaat Cloud Computing

1. Efesiensi Biaya

2. Skalabilitas

3. Aksesibilitas

4. Keamanan

5. Kolaborasi

Jenis Layanan Cloud

1. IaaS: Infrastruktur virtual

2. PaaS: Platform pengembangan aplikasi

3. SaaS: Aplikasi siap pakai
